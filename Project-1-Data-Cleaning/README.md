Project 1 – Data Cleaning & Data Preparation

Objective

To improve data quality by identifying inconsistencies, handling missing information, and preparing the dataset for further analysis.

Tasks Performed

Examined the dataset for missing and incomplete entries.

Replaced 309 empty Coupon Code values with "No Coupon".

Converted date values into a consistent format.

Standardized Unit Price and Total Price columns.

Verified price calculations for accuracy.

Inspected records for duplicate entries and unique ID issues.


Tools Used

Microsoft Excel


Outcome

Successfully cleaned and organized the dataset, ensuring accuracy, consistency, and readiness for analysis.

Status

✅ Completed
